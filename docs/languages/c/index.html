    Pointers

    int x = 1, y = 2, z[10];
int *ip;     /* ip -> int*/

ip = &x;     /* ip -> x */
y = *ip;     /* y = 1 */
*ip = 0;     /* x = 0 */
ip = &z[0];  /* ip -> z[0]*/

    При передаче в функцию, переменные копируются, поэтому для изменения данных во внешней функции нужно передавать указатели.
