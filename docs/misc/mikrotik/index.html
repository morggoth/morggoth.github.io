    Mikrotik

    NetInstall

    Netinstall tool originaly delevoped for Windows. I don`t test it via wine.
    1. Disable Firewall
    2. Connect PC to Port 1 on Mikrotik
    3. Disable all other network interfaces on the PC - LAN, Wireless, Virtualbox
    4. Set static IP of 192.168.88.3 subnet mask 255.255.255.0 gateway 192.168.88.1 on PC
    5. Run NetInstall (download page)
    6. Select “Net Booting”
    7. Mark “Boot Serve Enabled”
    8. Selected Client IP address of 192.168.88.1
    9. Keep reset pressed while powering on
    10. Keep holding the reset pin till a beep sounds. Release immediately
    11. Router showed up in Netinstall's list
    12. Unzipped “all_packages-mipsbe-X.X.zip”
    13. Selected need packages or all of them by browsing to the unzipped directory and using the “Select All” button
    14. Clicked on “Install”
    15. The progress bar moved to 100% as the packages were uploaded
    16. The router rebooted - Beep once, some time later (1 min or so if I recollect) a second beep
